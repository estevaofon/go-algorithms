## ğŸ“š Stack

A stack is a LIFO (Last In, First Out) data structure. The current implementation supports:

- `Push(value)` - Adds an element to the top
- `Pop()` - Removes and returns the top element
- `IsEmpty()` - Checks if the stack is empty
- `Size()` - Returns the stack size
